# 🍽️ LE PALAIS – Site Web de Restaurant

Un site web moderne et élégant réalisé avec HTML, CSS et JavaScript.

## 📌 Présentation du Projet

LE PALAIS est un site web de restaurant de luxe conçu pour offrir une expérience utilisateur moderne avec un design élégant, une interface responsive et des animations interactives.

Ce projet se concentre sur le développement Front-End avec une structure propre et une interface attractive.

---

## 🚀 Technologies Utilisées

- HTML5
- CSS3
- JavaScript

---

## 🎨 Fonctionnalités

✅ Design Responsive  
✅ Navbar Moderne  
✅ Hero Section avec logo élégant  
✅ Cartes interactives du menu  
✅ Galerie d’images  
✅ Animations Hover  
✅ Effets d’apparition au scroll  
✅ Expérience utilisateur fluide  

---

### 🏠 Section Accueil

### 🍴 Section Menu

- Plats principaux

### 📸 Galerie
Galerie moderne avec effets hover.

### 📍 Visit Us
Section contenant les informations et la localisation du restaurant.

## ⚡ Fonctionnalités JavaScript

- Animations Hover
- Effets Zoom sur les images

---

## 📱 Responsive Design

Le site est entièrement responsive et compatible avec :
- Ordinateurs
- Tablettes
- Smartphones

---

## 🎯 Objectif du Projet

L’objectif de ce projet est de pratiquer les compétences en développement web Front-End et de créer une interface moderne pour un site de restaurant en utilisant les technologies web modernes.
